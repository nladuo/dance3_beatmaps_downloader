# dance3_beatmaps_downloader
舞立方自制谱下载


### 1. 下载谱面列表
```shell
python 1_crawl_info_list.py
```


### 2. 下载谱面详情
```shell
python 2_crawl_info_detail.py
```


### 3. 设置谱面列表
```shell
python 3_make_beatmap.py
```


### 4. 下载谱面
```shell
python 4_download_beatmaps.py
```

### 其他
```shell
# 导出所有的信息
python dumps_infos.py
```
